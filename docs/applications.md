## MR-to-CT Image Synthesis

A reference implementation for MR-to-CT synthesis based on NV-Generate-CTMR (`rflow-ct`) is available here:
[https://github.com/brudfors/maisi-mr-to-ct](https://github.com/brudfors/maisi-mr-to-ct)

## Educational Toolkit for Radiotherapy Research

A radiotherapy-focused educational toolkit that bundles interactive Jupyter notebooks and DICOM RT-STRUCT export so synthetic CT + mask pairs from NV-Generate-CTMR are loadable in clinical tools (3D Slicer, RayStation, Eclipse) is available here:
[https://github.com/MustafaKadhim/MAISI-v2-Educational-for-Radiotherapy-Research](https://github.com/MustafaKadhim/MAISI-v2-Educational-for-Radiotherapy-Research)
